Hvad dælen er Git?

Git er noget vi skal bruge fordi det åbenbart er nemt? Not! Anyways. Her er nogle commandoer!

Git cd "folder"

Man skal huske at "connecte" sin Git mappe så det "Repository" man vil hente, ender det rigtige sted.

Det gør man først ved at åbne sin Comman Promt og skrive "cd (slip din git folder i vinduet)"

Så har Git selv fundet ud af hvor den skal smide alle de filer du henter.

Git Clone

Den første gang man henter en fil, skal man clone det

Det at clone en fil, vil sige at man henter en fil via en URL

Inde i commando prompt skriver man: git clone
